Installation

TableNav is a the jQuery plugin, used you need jquery before install it,then

<script src="path/to/jquery.tableNavigator.js"></script>
<script>

$(function() {
$("table").tableNavigator();
});

</script>

Examples

You lots of awesome thing using table, here some examples

1. Basic Example

                      


<script>

$(function() {
$("table").tableNavigator({
	input:true
});
});

</script>

2. Custom Background Color

                      

<script>

$(function() {
$("table").tableNavigator({
	background:'#89AAE6',
	input:true
});
});

</script>

3. Without Input
